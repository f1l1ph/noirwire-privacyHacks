## Notes

### On Noir + Solana Integration

The Noir toolchain natively generates Solidity verifiers for EVM chains. For Solana:
1. **Option A**: Generate Noir proof off-chain, use Groth16 BN254 compatible proof, verify on Solana using `alt_bn128_pairing` syscall
2. **Option B**: Use intermediate proof systems that can bridge Noir proofs to Solana-compatible formats
3. **Option C**: Generate proofs with Noir, verify using a custom Anchor program that implements the verification logic

See [solana-zk-proof-example](https://github.com/wkennedy/solana-zk-proof-example) for Groth16 on-chain verification patterns.
